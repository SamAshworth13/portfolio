This folder contains a set of small projects worked on during the Advanced Programming unit
of my second year of university. These were made using Java in Eclipse.

DungeonCrawler is a simple text-based game of chance where the player must choose whether or
not to fight a monster, without knowing how much damage it will deal to them.
The aim is to get the highest score possible without being defeated.

BarcodeGen is a program with a UI for generating barcodes of different standards, given a
string of characters.
